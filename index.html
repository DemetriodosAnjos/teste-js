Sintaxe

const name = 'Demetrio';
console.log(name);

O que é?

const é a palavra-chave em JavaScript usada para declarar uma constante. Diferente de let e var, o valor de uma const não pode ser alterado após a sua declaração.

O objetivo principal é garantir que a variável mantenha o mesmo valor durante toda a execução do código, evitando reatribuições acidentais e tornando o seu código mais seguro e previsível.
